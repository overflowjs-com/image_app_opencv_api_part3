
Getting Started
---------------

```sh
# clone it
git clone https://github.com/overflowjs-com/image_app_opencv_api_part3.git

# Make it your own
rm -rf .git && git init && npm init

# Install dependencies
npm install

# Start development live-reload server
PORT=4000 npm run dev

# Start production server:
PORT=4000 npm start
```
